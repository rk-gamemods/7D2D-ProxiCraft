[size=4][b]ProxiCraft v1.3.0[/b][/size]

Game Version 2.6 (build 22422060) compatibility update.

[color=#ff9900][b]⚠ Note:[/b][/color] This release has not been fully play-tested yet. It should be backwards compatible with game version 2.5, but if you run into issues, [b]v1.2.14 remains available[/b] as a safe fallback. Please report any problems so they can be fixed quickly.

[b]Changes:[/b]
[list]
[*][b]Updated dew collector item access[/b] - The game renamed TileEntityCollector.items to Items (uppercase). ProxiCraft now uses a version-tolerant accessor with multiple fallback strategies, so it works across both old and new game builds.
[*][b]Fixed dew collector UI not refreshing[/b] - When ProxiCraft removed items from a dew collector, the new XUiC_DewCollectorContainer UI window wouldn't update until closed and reopened. Now calls NotifyListeners() before SetModified() to match the game's own notification pattern.
[/list]

[b]Installation:[/b]
[list=1]
[*]Download ProxiCraft-1.3.0.zip
[*]Extract the zip into your Mods folder
[*]Launch game with [b]EAC disabled[/b]
[/list]

[b]Multiplayer:[/b] All players need the same version installed.

[b]Bug Reports:[/b] Comment on this mod page with pc_debug.log from your ProxiCraft folder.
