[size=4][b]ProxiCraft v1.2.8[/b][/size]

Multiplayer Thread Safety & Crash Prevention

[b]Fixed:[/b]
[list]
[*]Fixed race condition in client verification counter causing potential unlock failures
[*]Fixed potential crash when iterating container locks during player disconnect
[*]Fixed thread safety issues in item count cache that could cause rare crashes
[*]Fixed concurrent modification vulnerability in storage priority ordering
[*]Fixed GetStorageItems iteration without snapshot causing rare InvalidOperationException
[/list]

[b]New:[/b]
[list]
[*]Added FlightRecorder crash diagnostics system - captures last 50 events before crash
[*]Flight recorder output saved to ProxiCraft/flight_recorder.log on crash or clean shutdown
[/list]

[b]Technical Changes:[/b]
[list]
[*]Client count now uses atomic Interlocked operations for thread safety
[*]Container lock iteration now uses defensive ToArray() snapshots
[*]Storage ordering now returns defensive copy instead of modifying original
[*]Item count cache now uses dedicated lock object with proper synchronization
[/list]

[b]Installation:[/b]
[list=1]
[*]Download ProxiCraft-1.2.8.zip
[*]Extract to your 7 Days to Die Mods folder
[*]Result: 7DaysToDie/Mods/ProxiCraft/
[/list]

[b]Multiplayer:[/b] Install on server AND all clients (same version)
